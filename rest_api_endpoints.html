Data Readings

/data_readings get post

get

Gets data readings of the specified type sorted by sensor name ascending then by timestamp descending

post

Adds a data reading of the specified type

get /data_readings

Gets data readings of the specified type sorted by sensor name ascending then by timestamp descending

  • Request
  • Response

Query Parameters

  • type: required (one of soil, temp, humid, batt, fanspeed)

    the type of data reading

  • start_time: (integer)

    the earliest timestamp to retrieve of the data readings (default to 10 minutes ago)

  • end_time: (integer)

    the latest timestamp to retrieve of the data readings (default to current time)

HTTP status code 200

Body

Media type: application/json

Type: array of object

Items: DataReading

  • sensor: required (object)
    • type: required (one of soil, temp, humid, batt, fanspeed)
    • name: required (string)
  • timestamp: required (integer)

    the number of milliseconds since the Unix epoch

  • reading: required (number)

    The value of the data reading for the given type. Units are temperature: Fahrenheit, humidity: %, soil moisture: % water volume, fan speed: rpm

  • health: (one of good, low, critical)

    the health of the sensor if it has one

HTTP status code 400

Body

Media type: application/json

Type: object

Example:

{
  "message": "start_time and end_time must be integers"
}

post /data_readings

Adds a data reading of the specified type

  • Request
  • Response

Body

Media type: application/json

Type: object

Properties
  • sensor: required (object)
    • type: required (one of soil, temp, humid, batt, fanspeed)
    • name: required (string)
  • reading: required (number)

    The value of the data reading for the given type. Default units are temperature: Fahrenheit, humidity: %, soil moisture: % water volume, fan speed: rpm

  • unit: (one of temp_c, temp_f, soil_raw, soil_percent, humid_percent, batt_volts, fanspeed_rpm)

    The unit of the value given for the reading. Readings that are not relevant to the sensor type will become the default for that type

HTTP status code 201

HTTP status code 400

Body

Media type: application/json

Type: object

Example:

{
  "message": "type cannot be null"
}

Settings

/settings get post

get

Gets the settings

post

Updates the settings if they exist and creates them if they do not

get /settings

Gets the settings

  • Response

HTTP status code 200

Body

Media type: application/json

Type: object

Properties
  • is_manual_mode: required (boolean)
  • temperature: required (object)

    temperature in degrees Fahrenheit

    • min: required (integer)
    • max: required (integer)
  • soil_moisture: required (object)

    soil moisture in percent water volume

    • min: required (integer)
    • max: required (integer)
  • lights: required (object)

    time frame that the lights should be on

    • start_time: required (time-only)
    • end_time: required (time-only)
  • watering_times: required (array of time-only)

    times when plants will be watered if they need to be watered

  • error_flush_times: required (array of time-only)

    times when any pending errors should be sent out as a notification

  • email_addresses: required (array of string)
